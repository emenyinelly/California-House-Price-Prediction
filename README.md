#  Project Objective
Build and deploy a regression model to predict median house prices in California districts based on various features such as income, housing age, and location.

#  Features
- Comprehensive data exploration and visualization
- Feature scaling and preprocessing
- Training of Linear Regression and Random Forest models
- Model evaluation using MAE, RMSE, and R² Score
- Model saving for deployment
- Real-time house price prediction function

#  Technologies Used
- Python
- pandas, NumPy
- scikit-learn
- Matplotlib & Seaborn
- joblib (for model deployment)


# How to Run the Project
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/california-house-price-prediction.git
   cd california-house-price-prediction

2. Install dependencies:
   pip install pandas numpy scikit-learn matplotlib seaborn joblib

3. Open the jupyter notebook:
   jupyter notebook california_house_price_prediction.ipynb
